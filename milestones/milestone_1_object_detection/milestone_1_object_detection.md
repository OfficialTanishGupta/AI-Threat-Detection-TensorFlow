# Milestone 1: Object Detection Module

## Objective

Detect objects held by individuals in live or recorded video footage using a deep learning model.

## Approach

- Used YOLOv8 pretrained model
- Processed video frames
- Detected objects such as knives, bags, bottles, and persons

## Tools

- TensorFlow
- YOLOv8
- OpenCV
- Google Colab

## Output

Annotated video with bounding boxes and confidence scores.
